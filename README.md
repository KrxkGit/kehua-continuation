# 时刻 (Kehua Moment) - 延续可话

**时刻** 是一款专注于私人记录、轻量且可高度自定义的动态记录工具。它旨在延续“可话”的纯粹记录体验，通过模块化架构确保在 WebStorm 等 IDE 中的高效维护。

## 🌟 核心特性

* **模块化维护**：将原本臃肿的单文件拆分为 `storage`, `ui`, `cloud`, `visual`, `app` 五大模块，逻辑清晰，互不干扰。
* **双向联动时间轴**：侧边栏时间轴支持年、月、日层级展开。点击可跳转，页面滚动时时间轴自动高亮并同步跟随。
* **搜索与高亮**：支持实时搜索过滤，并对搜索结果进行自动关键词黄色高亮处理。
* **云端备份与加密**：支持将记忆通过 AES 加密后推送至 GitHub Gist，保护个人隐私。
* **AI 记忆星图**：自动生成互动词云（星图），支持点击词汇快速定位或屏蔽特定词汇。
* **AI 全量总结**：接入 AI API，一键对历史动态进行精炼总结。
* **多维配置导入**：支持手动保存、二维码导出、摄像头扫码及本地图片识别配置信息。
